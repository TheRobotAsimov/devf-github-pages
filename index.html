¿Qué es GitHub Pages?

GitHub Pages es un servicio gratuito de GitHub para alojar sitios web estáticos directamente desde un repositorio. Es ideal para páginas personales, documentación, blogs simples y demos.

¿Para quién es útil?

  • Desarrolladores que quieren publicar demos o portafolios rápidamente.
  • Proyectos que necesitan documentación alojada junto al código.
  • Usuarios que desean un sitio estático sin gestionar servidores.

Características principales

  • Alojamiento gratuito desde tu repositorio GitHub.
  • Soporte para dominios personalizados y HTTPS.
  • Integración con Jekyll para generar sitios estáticos si lo deseas.

Cómo empezar (rápido)

Paso 1
Crear un repositorio público o usar la rama `gh-pages`.
Paso 2
Agregar los archivos estáticos (`index.html`, CSS, imágenes).
Paso 3
Ir a la configuración del repositorio y activar GitHub Pages.
Paso 4
Opcional: configurar un dominio personalizado y HTTPS.
