# devf-github-pages

Página web estática creada como proyecto para el bootcamp DEV.F, publicada con **GitHub Pages**.

## Descripción

Sitio informativo en español sobre **¿Qué es GitHub Pages?**: qué es, para quién es útil, sus características principales y los pasos para empezar a usarlo.

## Características

- Estructura semántica HTML5 (`header`, `main`, `footer`, listas, párrafos, imágenes, enlaces)
- Diseño dark-mode con CSS puro (sin frameworks externos)
- Totalmente responsivo (CSS Grid con `auto-fit`)
- Sin dependencias de JavaScript

## Tecnologías

- HTML5
- CSS3 (variables CSS, Grid)
- Git / GitHub / GitHub Pages

## Cómo desplegar (GitHub Pages)

1. Haz un fork o clona el repositorio:
   ```bash
   git clone https://github.com/TheRobotAsimov/devf-github-pages.git
   ```
2. Empuja tus cambios a la rama `main`.
3. En la configuración del repositorio (`Settings > Pages`), selecciona la rama `main` y la carpeta raíz (`/`).
4. GitHub generará automáticamente la URL pública.
